# Retail Sales SQL Analysis

## 📌 Project Overview
This project analyzes retail sales data using SQL to extract meaningful business insights.

## 🛠 Tools Used
- MySQL
- GitHub

## 📊 Dataset
Sample Superstore Dataset

## 🔍 Key Analysis Performed
- Total Sales Calculation
- Top 5 Products by Revenue
- Products Above Average Price
- Sales Aggregation using GROUP BY
- Business KPI Analysis

## 🎯 Business Insights
- Identified top revenue-generating products
- Compared product prices with average price
- Analyzed overall sales performance

## 📁 Files in Repository
- `analysis.sql` → Contains all SQL queries
- `Sample_Superstore.csv` → Dataset used
